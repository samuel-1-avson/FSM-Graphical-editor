Quick Start Guide - BSM Designer

Welcome to the Brain State Machine Designer!

Basic Operations

  • Adding States: Use the "Add State" tool or drag a state from the "Elements" dock.
  • Adding Transitions: Use the "Add Transition" tool and click on the source state, then the target state.
  • Editing Properties: Double-click an item or select it and click "Edit Properties..." in the "Item Properties" dock.
  • Saving/Loading: Use the File menu (File > Save, File > Open).

Python Simulation

Use the "Python Simulation" dock to start, stop, reset, and step through your FSM. Trigger events and observe variable changes.

AI Assistant (Gemini)

Configure your Google AI API Key in "AI Assistant" > "AI Assistant Settings (Gemini)...". You can then chat with the AI or ask it to generate FSMs from descriptions.

Preferences

Customize the application's appearance and behavior via "Edit" > "Preferences...". You can change the theme (Light/Dark), canvas colors, and default visual styles for new FSM elements.
